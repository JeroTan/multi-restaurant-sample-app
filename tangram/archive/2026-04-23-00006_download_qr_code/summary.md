# Feature Summary: Downloadable QR Codes

## Intent
To provide restaurant administrators with the ability to download individual table QR codes as image files directly from the Tables & QRs dashboard. This adds utility beyond the bulk print feature, allowing them to save or share specific QR codes.

## Scope
- **UI Enhancement:** Add a "Download" button to each individual QR code card in the `TablesAdminClient` component.
- **Client-Side Generation:** Utilize browser APIs to convert the SVG generated by `react-qr-code` into a downloadable PNG image.

## Strategic Fit
This feature directly fulfills the remainder of **FR-06 (QR Generation)** from the Business Requirements Document, which specifies that QR codes should be downloadable. It enhances the admin workflow by allowing for more granular QR code management and distribution.

## Execution Log
- Pace: All-at-Once
- Modified `src/components/TablesAdminClient.tsx` to include the `Download` icon from `lucide-react`.
- Added the `downloadQR` function which serializes the generated SVG QR code into a data URL, paints it onto an HTML5 `<canvas>` (with padding), and triggers a PNG download natively in the browser.
- Included the new "Download PNG" button on each table card, ensuring it is hidden during bulk printing via the `print:hidden` Tailwind utility class.
